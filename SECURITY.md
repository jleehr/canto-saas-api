# Security Policy

## Supported Versions

Only the latest release of `jleehr/canto-saas-api` receives security fixes.

## Reporting a Vulnerability

Please report vulnerabilities privately via GitHub's security advisory form:

https://github.com/jleehr/canto-saas-api/security/advisories/new

Do **not** open a public issue for security problems. You will receive a
response as soon as possible; please allow a reasonable time for a fix and
coordinated disclosure before publishing details.
